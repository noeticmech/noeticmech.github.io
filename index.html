Matthew Butler

A noetic mechanic working on thinking machines

Welcome my personal website. I am a recent graduate of Computer Science doctoral program at the University of Tulsa. Most of my research focus has been on Systems-level Information Assurance, and Secure Design, with some substantial dabbling into Statistical learning and more classic Artificial Intelligence. I also have some marginal interest in adjacent fields like linguistics, economics, and psychology.

I like to typeset and bind custom books, am slowly learning to sing Byzantine chant, and enjoy gardening and hiking.

I primarily work with Linux (most experience with Slackware and Arch Linux), Python (including Numpy, Scipy, and Scikit-learn), Eclipse and VS Code, LaTeX and Pandoc. Most of my classwork was done in Java and C++, and some of my research has required C.

Projects

  • truck-ids. Part of my doctoral work in intrusion detection on J1939 networks. I'm slowly refactoring my project code to reflect my current, better understanding of pythonic idiom. The original code is very Java-esque.

  • marginalia. The theme I created for this site, based on Tufte CSS. I really enjoy the aesthetics of prayer books and medieval manuscripts and marginalia, and have attempted to replicate some of that for this site.

  • prayerbook. I've found a lot of prayerbooks out there to be lacking in size and content, so I made my own for my own personal use. I know other people are interested in doing something similar, so I'm sharing this project so others can see how I've formatted and organized mine.
